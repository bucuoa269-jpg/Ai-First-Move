# AI先机 V2｜构建与验收报告

生成日期：2026-06-18

## 1. 完成范围

- 首页、机会流、稳副业、创业实验、找工作、比赛雷达
- 10个城市索引与城市详情页
- 时间机器、全球视野、8个专题页
- 40条机会详情页与来源账本
- 3条深度访谈案例
- 信源说明、产品说明、本地收藏与搜索历史
- 免登录、LocalStorage、语音输入、TXT/MD读取

- 独立分享链接、Web Share、1:1与9:16分享卡预览
- 浅色默认、深色模式、移动端底部导航

## 4. 运行方式

开发模式：

```bash
npm install

```

打开 `http://localhost:3000`。

```bash
python -m http.server 4173 -d out
```

打开 `http://localhost:4173`。

Windows用户也可以双击：

- `启动开发版.bat`
- `启动静态预览.bat`

## 5. AI能力边界

当前无需API即可使用：

- 用户需求关键词解析
- 标签与权重推荐
- Fuse.js模糊搜索
- 城市、类型、技能和信源匹配

需要未来服务端API才能增强：

- 更复杂的自然语言结构化
- 上传PDF/DOCX正文解析
- AI解释推荐理由的动态生成
- 定期采集、去重和后台审核

模型必须只能引用数据库里已有的机会与链接。

## 6. 部署

- Vercel：导入GitHub仓库即可构建；网址在项目的 Domains 区域。
- GitHub Pages：工作流已位于 `.github/workflows/deploy-pages.yml`；部署成功后网址在仓库 Settings → Pages 顶部。
- 当前沙盒没有用户的GitHub/Vercel授权，因此没有替用户发布真实公网域名。

## 7. 人工核验

详见 `docs/manual-verification.md`。最优先核验：

- HarmonyOS两类赛事的最新截止与资格
- B站AI创作活动当前状态与版权条款

- MICCAI GAVE2提交日期

## 8. 已知限制

- PDF/DOCX上传当前只提供入口和友好提示，纯前端版不解析正文。
- 首批城市页是信息架构与已核验机会索引，尚未达到每城10家公司、10个岗位的运营目标。
- 深度访谈中的收入均按当事人自述处理，不代表普遍结果。
