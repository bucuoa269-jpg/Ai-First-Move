# 时间机器与机会迁移方法

时间机器不是照搬，也不是把地区描述成“落后”。它研究不同市场、城市、行业和平台的成熟度差异。

## 五条迁移轴

1. 全球成熟市场 → 中国市场
2. 一线城市 → 二三线城市
3. 中国成熟模式 → 东南亚、非洲及其他新兴市场
4. 科技行业 → 传统行业
5. 平台新能力 → 服务、工具或产品

## 每次迁移必须回答

- 原市场有什么成立证据？
- 目标市场出现了什么真实信号？
- 谁付款，为什么付款？
- 支付、渠道、文化、监管、供应链和履约有什么差异？
- 7天内能做什么最小实验？
- 什么结果意味着应该停止？

只有获得访谈、预售、定金、真实使用或复购证据，才能把“启发”升级为“机会”。
