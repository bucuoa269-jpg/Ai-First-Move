# AI先机编辑与审核政策

## 核心原则

1. 国内机会为主，全球信息用于参照和机会迁移。
2. 官方入口永远免费展示，不把公开链接藏在付费墙后。
3. 不保证收益，不使用“稳赚、躺赚、轻松月入”等表达。
4. 事实、推断、建议分开标注。
5. 社交媒体和播客用于发现线索，涉及日期、资格、岗位和奖励时应回到官方页面。

## 发布流程

线索发现 → 字段提取 → 去重 → 找确认来源 → 编辑摘要 → 人工核验 → 发布 → 到期复核 → 归档。

## 时效规则

- 招聘：7天内优先；超过30天要求重新核验。
- 比赛：每天计算截止状态；提交前再次提醒用户打开官方页面。
- 平台激励：3天内优先。
- 深度案例：保留原始发生时间，不以新上传时间伪装成新案例。
- 编辑分析：每30天检查需求信号；没有新证据时降低推荐分。

## 商业内容

赞助、广告和返佣必须醒目标记，不改变信源等级和编辑评分。禁止向学生销售高价“稳赚项目”。
